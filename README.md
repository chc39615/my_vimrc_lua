# vimrc lua script

## require
- Neovim 0.7+


## installation
1. git clone to ~/.config/nvim
```bash
git clone ~/.config/nvim
```
2. start nvim, let the script do the rest

3. if "can not find colorscheme occurs", just run :PackerSync  
to download the files
